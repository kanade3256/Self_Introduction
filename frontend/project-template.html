研究・データサイエンス

歩行データ時系列クラスタリング研究

MediaPipeで骨格データ抽出後、時系列補完（スプライン/PCHIP）→DTW＋クラスタリングにより歩行スタイルを分類

期間 2023年9月 - 2024年3月
役割 研究・実装・分析
技術スタック Python, pandas, SciPy, scikit-learn, DTW, k-means
研究レポート ポスター GitHub

背景・課題

歩行データの解析は、リハビリテーション・スポーツ科学・ヘルスケア分野で重要な研究テーマです。 従来の歩行解析は高価な専用機器や主観的な評価に依存しており、より手軽で定量的な解析手法が求められていました。

  • 専用機器による高コスト・アクセス性の問題
  • 主観的評価による再現性・客観性の課題
  • 歩行スタイルの定量的分類手法の不足

アプローチ・技術

1

データ取得

MediaPipeを使用して動画から骨格データを抽出

MediaPipe, OpenCV
2

前処理・補完

欠損データの補完（スプライン/PCHIP補間）とノイズ除去

SciPy, pandas
3

特徴量設計

歩行周期・関節角度・速度パターンの特徴量抽出

NumPy, 信号処理
4

距離計算・クラスタリング

DTW（Dynamic Time Warping）による時系列距離計算とk-meansクラスタリング

DTW, scikit-learn

成果・結果

定量的分類の実現

歩行パターンを客観的指標で3つのクラスターに分類することに成功

手軽なシステム構築

一般的なWebカメラを使用した低コスト・高アクセス性のシステムを構築

研究発表

学内研究発表会でポスター発表を実施、研究手法の有効性を実証

技術的なポイント

  • 時系列データ処理: 不規則なサンプリング間隔やノイズを考慮した前処理パイプライン
  • DTWアルゴリズム: 歩行周期の個人差を考慮した類似度計算手法
  • 特徴量エンジニアリング: ドメイン知識を活用した生体力学的特徴量の設計
  • 可視化・検証: クラスタリング結果の解釈性を高める可視化手法の実装

学び・今後の展望

学んだこと

  • 実データを扱う際の前処理・ノイズ処理の重要性
  • ドメイン知識と機械学習技術の融合アプローチ
  • 研究における仮説設定・検証・考察のプロセス
  • 時系列データ解析の専門的手法（DTW、補間など）

今後の展望

  • リアルタイム解析システムの構築
